> **TPS Note**: computed as `generated_tokens / wall_time`

## SQ Sub-metrics

| Metric | Weight | Description |
|--------|:------:|-------------|
| structural_char_F1 | 0.4 | F1 of structural characters ({,},[,],(,),:,,,=,\n) between pred and ref |
| bracket_balance_score | 0.3 | 1.0 if all (), [], {} are balanced, 0.0 otherwise |
| structure_type_preservation | 0.2 | 1.0 if output preserves expected structure type |
| no_repetition_score | 0.1 | 1.0 - 3×repetition_rate, penalizes repeated blocks |
